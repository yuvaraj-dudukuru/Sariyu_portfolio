# Somesula Sariyu - Personal Portfolio

A fully responsive, modern personal portfolio website built with HTML, CSS, and JavaScript.

![Portfolio Preview](https://placehold.co/1200x630/1f242d/0ef?text=Sariyu+Portfolio)

## 📌 Project Overview

This repository contains the source code for the personal portfolio of **Somesula Sariyu**, a Cybersecurity Analyst & Developer. The design is clean, highly interactive, and fully responsive across all devices (mobile, tablet, laptop, and desktop).

## 🚀 Features

- **Modern UI/UX**: Dark theme with a sleek, glowing cyan accent (`#0ef`).
- **Responsive Design**: Pixel-perfect layouts for all screen sizes using CSS Grid, Flexbox, and Media Queries.
- **Scroll Animations**: Smooth reveal animations while scrolling, powered by [ScrollReveal.js](https://scrollrevealjs.org/).
- **Typing Effect**: Dynamic typing text effect in the Hero section using [Typed.js](https://github.com/mattboldt/typed.js/).
- **Sticky Navigation**: A sticky header with active link highlighting as you scroll through sections.
- **Project Showcase**: A dedicated section to display the latest projects with tech stack details and links.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom properties, animations, and responsive media queries.
- **JavaScript (Vanilla)**: DOM manipulation, event listeners, and interaction logic.
- **BoxIcons**: Scalable vector icons for UI elements.

## 🌐 Deployment Instructions

This project is fully compatible with **GitHub Pages**.

1. Fork or clone this repository to your local machine:

   ```bash
   git clone https://github.com/yuvaraj-dudukuru/Sariyu_portfolio.git
   ```

2. Make sure all paths (e.g., `css/style.css`, `assets/profile.jpg`, `js/script.js`) are kept relative. Do not use leading slashes (`/`) so that the subpath deployment works seamlessly.
3. Push your changes to your `main` branch.
4. Go to **Settings** > **Pages** on your GitHub repository.
5. Under "Build and deployment", select "Deploy from a branch" and choose `main`.
6. Save and wait for the deployment process to complete. Your site will be live at `https://your-username.github.io/Sariyu_portfolio/`.
