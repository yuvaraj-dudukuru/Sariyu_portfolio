# Somesula Sariyu | Cybersecurity Analyst & Full-stack Developer

![Sariyu Portfolio Screenshot](https://placehold.co/1200x630/0a0e17/00abf0?text=Sariyu+Portfolio)

Professional portfolio of Somesula Sariyu, a Senior Cybersecurity Analyst and Full-stack Developer specializing in vulnerability management, security analysis, and building secure, modern web applications.

## 🚀 Key Features

- **Modern Glassmorphism UI**: A premium, recruiter-grade design with a dark/light mode toggle.
- **Project Filtering**: Interactive filtering by technology stack.
- **GitHub API Integration**: Real-time project stats (stars/forks) fetched directly from GitHub.
- **Responsive Design**: Seamless experience across all devices.
- **SEO & Accessibility Optimized**: Semantic HTML5, ARIA labels, and OpenGraph metadata.
- **Performance Focused**: Minimal dependencies, lazy-loaded images, and optimized assets.

## 🛠️ Built With

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3 (Vanilla)**: Modern features like Glassmorphism, CSS Grid, and Custom Properties.
- **JavaScript (Vanilla)**: Modular, dependency-free interaction and API integration.
- **Boxicons**: For crisp, high-quality iconography.
- **Google Fonts**: Premium typography (DM Sans & JetBrains Mono).

## 📁 Repository Structure
